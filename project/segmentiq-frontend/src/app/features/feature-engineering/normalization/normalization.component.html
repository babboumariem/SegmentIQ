📐 ML Pipeline Formulas · from Raw Data to Business Insights
Raw DataCustomers · Orders · Products · Returns — 4 CSV tables imported into SQLite database
Data CleaningDuplicate removal · null filling · type conversion · anomaly filtering
Feature Engineeringtotal_spend · order_count · recency · aov · spend_velocity · return_rate · diversity · clv_slope
Data PreprocessingWinsorization clip(p01, p99) on all features + StandardScaler → z = (x − μ) / σ → normalized matrix Xs
K-Means Clusteringfit_predict(Xs) · K clusters · n_init=10 · random_state=42 — applied on Xs
Customer SegmentationSorted desc by avg total_spend → 0=VIP · 1=Loyal · 2=At Risk · 3=Lost · business labels
PCA VisualizationX_pca = Xs · Vᵀ · applied on Xs AFTER K-Means · 2D only · n_components=2 · random_state=42
